RSA (Rivest-Shamir-Adleman)

  • Key Distribution: Asymmetric nature provides strong security
  • Key Strength: 2048-bit keys provide adequate security for current computing capabilities
  • Limitation: RSA can only encrypt data up to (key size - padding) bytes. For 2048-bit keys, maximum encryption capacity is approximately 245 bytes
  • Best Usage: Suitable for encrypting small amounts of data or key exchange
