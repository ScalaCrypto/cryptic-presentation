Encrypt


type Encrypt = PlainText => CipherText
    

case class CipherText(bytes: Array[Byte])
    

case class PlainText(bytes: IArray[Byte],
                     manifest: IArray[Byte])
  • The Encrypt type is simply a function from PlainText to CypherText
  • → CipherText simply holds the encrypted data.
  • → PlainText holds the unencrypted data along with an optional Manifest
  • Put the secrets in the bytes, anything else in the manifest (which will not be encrypted).
  • Knowledge about the structure of what is encrypted makes forced decryption easier.
