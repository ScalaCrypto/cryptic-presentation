RSA (Rivest-Shamir-Adleman)

Large keys

Small data

Suitable for key exchange

RSA was invented in 1977 by Ron Rivest, Adi Shamir and Leonard Adleman
  • → It needs large keys, minimum 2048-bit, to be considered secure, at least for todays computers
  • → It can only handle small data
    max data size is (key size - padding)
    For example, a 2048-bit key handles approximately 245 bytes of data
  • → It's suitable to use for key exchange
