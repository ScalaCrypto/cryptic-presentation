RSA (Rivest-Shamir-Adleman)

Large keys needed

Small data

Suitable for key exchange

Invented in 1977 by Ron Rivest, Adi Shamir and Leonard Adleman
→ It needs large keys, minimum 2048-bit
to provide adequate security for current computing capabilities
→ Max data size is (key size - padding)
2048-bit handles approximately 245 bytes
→ It's suitable to use for key exchange
