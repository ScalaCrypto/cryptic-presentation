    val aesKey = Aes.key(aesKeyBytes)

    val text = Aes.decrypt(textBytes, aesKey, ivSpec,

  • Here we split the ciphertext into parts.
  • Then decrypt the AES key bytes with the given private key
  • Then we recreate the AES key from the decrypted bytes
  • Finally we use the AES key to decrypt the encrypted text bytes.
