Types of Ciphers
Ciphers are either Symmetric or Asymmetric
What is the difference between the two types?
Symmetrical

same key for encryption/decryption

key distribution problem

But how do you send the secret key to the recipient?
This is known as the key distribution problem
A solution to this is Asymmetric ciphers
Asymmetrical

private/public keys

Limited data

  • Private - public key
  • The public is used for encryption
  • and the private for decryption
  • limited data size few hundred bytes
  • Hybrid, asymmetric for key exchange
