Cipher Types

  • Ciphers are either Symmetric or Asymmetric
  • But what is the difference between the two types? →

Symmetric

Same key for encryption/decryption

Fast and efficient

Key distribution problem

  • Symmetric cryptos use the same key for both encryption and decryption →
  • They are fast and efficient →
  • But they have a key distribution problem →
  • How do we securely share the key with the other party?
  • For that we use an asymmetric crypto →

Asymmetric

Private - public key pair

Public key encrypts

Private key decrypts

Limited data

  • The asymmetric crypto solves the key distribution problem by having a
    private - public key pair →
  • The public key is used for encryption →
  • And the private key for decryption →
  • A drawback is that you can only encrypt limited amount of data (a few hundred bytes) →
  • To solve this problem, we can use a hybrid scheme →

Hybrid

Generate a symmetric key

Encrypt payload using that symmetric key

Encrypt the symmetric key with a public key

  • In a hybrid scheme, you generate a symmetric key →
  • You then encrypt the payload with that symmetric key →
  • Finally, you encrypt the symmetric key with a public key →
  • This way, you can encrypt large amounts of data efficiently
