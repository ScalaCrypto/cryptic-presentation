How to keep your secrets from leaking
Encryption
Use the extension method 

val enc: Encrypted[String] = "secret".encrypted
Elliptic curve 

import cryptic.{*, given}
import cryptic.crypto.EC.given

val keyPair: KeyPair = EC.keygen(256)
given publicKey: PublicKey = keyPair.getPublic

val enc = "secret".encrypted
enc.toString // CipherText(0x04e5da5206...)
Decryption Use the decrypted method 

val decrypted: Try[String] = enc.decrypted
Elliptic curve 

import cryptic.{*, given}
import cryptic.crypto.EC.given

given privateKey: PrivateKey = keyPair.getPrivate

enc.decrypt // Success("secret")
