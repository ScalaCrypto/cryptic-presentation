Encryption

PlainText => CipherText


trait Encrypt:
  def apply(pt: PlainText): CipherText
So, what are PlainText and CipherText?
PlainText holds the unencrypted data.
CipherText holds the encrypted data.
How to create PlainText?
Codec 

trait Codec[V]:
  def encode(v: V): PlainText
  def decode(pt: PlainText): Try[V]

Provided

String, Array[Byte], Float, Int, Long, Double, Option[V]
To create to and from PlainText we use a Codec

Elliptic Curve 

given encrypt(using key: PublicKey): Encrypt

Elliptic Curve encryption 

import cryptic.{*, given}
import cryptic.crypto.EllipticCurve.given
import java.security.*

val keyPair: KeyPair = EllipticCurve.keygen(256)
given publicKey: PublicKey = keyPair.getPublic
val enc = "secret".encrypted
// ==> Encrypted(CipherText(0x04e5da5206...))
Elliptic Curve decryption 

import cryptic.{*, given}
import cryptic.crypto.EC.given
import java.security.PrivateKey

given privateKey: PrivateKey = keyPair.getPrivate
enc.decrypt
// ==> Success("secret")
