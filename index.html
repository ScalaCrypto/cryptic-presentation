Encryption

type Encrypt = PlainText => CipherText
So, what are PlainText and CipherText?
PlainText holds the unencrypted data.
CipherText holds the encrypted data.
How to create PlainText?
Codec 

trait Codec[V]:
  def encode(v: V): PlainText
  def decode(pt: PlainText): Try[V]
Provided

String, Array[Byte], Float, Int, Long, Double, Option[V]
To create to and from PlainText we use a Codec
Out of the box Cryptic provides Codecs for
Note about encrypting

Only encrypt the secrets


case class PlainText(bytes: Array[Byte],
                     manifest: Manifest = Manifest.empty)
Make sure that only the secret is encrypted
If an adversary knows something about what's encrypted, decryption is easier
When creating a Codec
use the PlainText Manifest to store structure.
The Manifest is not to be encrypted.
CipherText

case class CipherText(bytes: Array[Byte])
The result of encryption function is CipherText
The encrypted method

extension [V: Codec](value: V)
  def encrypted(using encrypt: Encrypt): Encrypted[V] =
                Encrypted(value)
We have an extension of the Codec
After encrypted we have CipherText wrapped in an Encrypted

case class Encrypted[V: Codec](cipherText: CipherText) extends
            Cryptic[V]

object Encrypted:
  def apply[V: Codec](value: V)(using encrypt: Encrypt):
        Encrypted[V] =
    if value == null then empty
    else Encrypted[V](encrypt(summon[Codec[V]].encode(value)))

AES

given encrypt(using passphrase: AESPassphrase,
              aesParams: AESParams): Encrypt

given decrypt(using passphrase: AESPassphrase,
             aesParams: AESParams): Decrypt
As AES is a symmetrical crypto the encrypt/decrypt methods are very similar
But the security lies in the AESParams
Default AES parameters use GCM mode

  case class GcmParams(
      factoryAlgorithm: String = "PBKDF2WithHmacSHA256",
      mode: String = "GCM/NoPadding",
      keyspecIterationCount: Int = 310000,
      keyspecLength: Int = 256,
      saltLength: Int = 32,
      version: Int = 1
  ) extends AesParams
Key derivation function - PBKDF2 with SHA-256 mode: String = "GCM/NoPadding", Number of iterations for key derivation (min 310000 per OWASP) Key length in bits (256 for Aes-256) Salt length in bytes (min 16 bytes recommended) Algorithm version for future migrations version: Int = 1 )

import cryptic.{given,*}

Elliptic Curve Integrated Encryption Scheme
  • Key Size Advantage: 256-bit ECC keys provide equivalent security to 3072-bit RSA
  • Performance: Faster key generation and smaller key sizes than traditional RSA
  • Security Features:
    • Hybrid encryption combining ECC with symmetric encryption
    • Built-in key agreement protocol (ECDH)
    • Message authentication (MAC) for integrity
  • Recommended Usage: Ideal for mobile and IoT applications where resources are constrained
